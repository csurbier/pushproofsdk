# iOS NSE setup (deprecated in-repo copy)

The **canonical, up-to-date guide** lives on the Pushproof website:

- **French:** [pushproof.dev/docs/#ios-nse](https://pushproof.dev/docs/#ios-nse)
- **English:** [pushproof.dev/en/docs/#ios-nse](https://pushproof.dev/en/docs/#ios-nse)

It covers target naming (`PushproofNotificationExtension`, not `PushproofNSE`), Swift Package
linkage, App Groups, `PushproofAppGroup` in Info.plist, and backend `mutable-content`
requirements.

For a quick checklist, run:

```bash
npx pushproof-install-nse
```
